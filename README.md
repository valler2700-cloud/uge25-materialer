# Uge 25 – undervisningsmaterialer (8B & 8D)

Lette, fravær-robuste materialer til en eksamensuge i matematik, historie og
religion (8. klasse). Forankret i Fælles Mål.

🌐 Spil/forside: https://valler2700-cloud.github.io/uge25-materialer/

## Mappestruktur
- `index.html` — forsiden med spil og download (ligger i roden).
- `Religion/` — religionsforløb om ondskab (naturlig, moralsk og banal ondskab + etik), etiske dilemmaer som gruppeopgave, og elevarket til ondskab.
- `Historie/` — scenariebaseret rollespil om 2. verdenskrig.
- `Matematik/` — escape-room-elevark til print (rumgeometri).
- `Faelles/` — de tværfaglige spil: Jeopardy (smartboard-quiz i to runder) og `Kortspil_religion-historie.html` (100 karakter-kort, Trumf-dyst og Tidslinje) plus `Elevark_Karakter-kamp`.
- `img/` — billeder, som Word-filerne bygges med.
- `build_docx.py` — bygger alle `.md` (også i fagmapperne) til matchende `.docx`.

Hver fagmappe rummer både den færdige `.docx` og dens `.md`-kilde.

> 🔒 Lærervejledninger og facit ligger bevidst ikke i dette offentlige repo. De er kun til læreren (lokale kopier). Det gælder escape-room-facit (`Escape-room_matematik_LAERER`) og lærervejledningerne til etik og Karakter-kamp.

## Brug
Åbn `index.html` (eller Pages-linket) og spil eller hent. Word-filerne kan printes og rettes frit.
